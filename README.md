# The Obsidian Assembly - Clone

## 📌 Descripción

Este proyecto es una recreación de la página web **The Obsidian Assembly** realizada con fines educativos y de práctica. **No es el sitio web oficial ni está afiliado de ninguna manera a The Obsidian Assembly o sus propietarios.**

El objetivo de este proyecto es demostrar mis habilidades en desarrollo web, diseño de interfaces y reproducción de experiencias visuales modernas utilizando tecnologías web actuales.

## 🎯 Propósito

Este proyecto fue desarrollado como:

* Práctica personal para mejorar mis conocimientos de desarrollo frontend.
* Ejercicio de análisis y reproducción de interfaces profesionales.
* Pieza de portafolio para mostrar mis capacidades técnicas y de diseño.
* Demostración de que puedo construir sitios web con una apariencia y experiencia de usuario similares a proyectos comerciales de alta calidad.

## ⚠️ Aviso Legal

Todo el contenido original, diseño, marca, logotipos y conceptos pertenecen a sus respectivos propietarios.

Esta versión fue creada únicamente con fines educativos y de demostración técnica. No tiene fines comerciales ni pretende suplantar al proyecto original.
